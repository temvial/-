# Интерактивная матрица приоритизации

## Описание проекта
Веб-приложение для анализа и приоритизации бизнес-процессов на основе двух групп критериев:

- Операционные 
- Стратегические 

Приложение автоматически рассчитывает средние значения критериев и распределяет процессы по зонам приоритета в матрице.

## Основные возможности

### Добавление новых процессов
Пользователь может создать новый бизнес-процесс через форму ввода и указать значения критериев по шкале от 1 до 5.
Для каждого процесса вычисляются:

- Средняя операционная эффективность
- Средняя стратегическая важность
- Приоритетная зона

### Визуальная матрица приоритетов
Все процессы отображаются в матрице из 4 зон:

1. Срочные изменения  
2. Развитие  
3. Стандартизация  
4. Мониторинг

### Таблица процессов
Данные выводятся в виде таблицы с возможностью:

- сортировки по столбцам
- удаления процессов
- просмотра рассчитанных показателей

### Фильтрация
Можно отобразить процессы только выбранной зоны.

### Сохранение данных
Все изменения автоматически сохраняются в localStorage браузера.

### Экспорт данных
Доступен экспорт списка процессов в формате JSON.

### Сброс данных
Можно вернуть исходный набор процессов.
